GitHub Hands-on Learning

This project is created to practice Git and GitHub concepts through hands-on implementation. It focuses on learning version control, repository management, branching, commits, pull requests, and collaborative development workflows.

Practice Objectives

  • Initialize a Git repository
  • Track project files
  • Create meaningful commits
  • Manage Git branches
  • Push code to GitHub
  • Pull latest changes
  • Create Pull Requests
  • Merge branches successfully
  • Resolve merge conflicts
